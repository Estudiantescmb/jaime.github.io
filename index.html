¿Es posible crear una ventana modal solo con HTML, sin necesidad de usar Javascript? ¡Por supuesto! Y aquí tenemos un ejemplo.

¡Esto es una ventana modal!

Y no le falta detalle. Su contenido, su botón parar cerrar, su fondo con color personalizado... ¡Tiene de todo!

Hablando del botón para cerrar, vamos a hacer clic en él para ver si todo funciona como debería.

Construir una ventana modal solo con HTML es muy sencillo. Solo es necesario saber que atributos y valores hay que aplicar a cada uno de los elementos.

Incluso podemos tener varios botones, cada uno de ellos enlazados a diferentes ventanas modales, con diferentes contenidos.
